# Vincent1994Go Home Assistant Add-ons

这是一个自定义的 Home Assistant 插件仓库，包含各种实用的插件。

## 如何添加到 Home Assistant

1. 打开 Home Assistant 管理界面
2. 进入 **Supervisor** > **加载项商店**
3. 点击右下角的三个点，选择 **仓库**
4. 点击右上角的 **添加**
5. 输入仓库 URL：`https://github.com/Vincent1994Go/addons`
6. 点击 **添加**，然后刷新页面

## 可用插件

### netease-cloud-music-api-enhanced
- **描述**：增强版网易云音乐 API
- **功能**：提供完整的网易云音乐 API 服务
- **版本**：1.0.0
