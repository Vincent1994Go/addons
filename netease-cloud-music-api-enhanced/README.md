# 网易云音乐API增强版 Home Assistant Add-on 存储库

本存储库包含网易云音乐API增强版的Home Assistant Add-on。

## 包含的Add-on

### 网易云音乐API增强版

全网最全的网易云音乐API接口，作为Home Assistant Add-on运行。

#### 功能特性

- 支持几乎所有网易云音乐API接口
- 支持音乐搜索、播放、下载
- 支持歌单、专辑、艺术家信息获取
- 支持用户登录和个人信息管理
- 支持音乐解锁功能
- 支持CORS跨域请求

## 如何添加本存储库

1. 在Home Assistant中，进入"配置" -> "加载项、备份与Supervisor"
2. 点击右上角的三个点，选择"存储库"
3. 添加以下URL：`https://github.com/Vincent1994Go/api-enhanced-main`
4. 回到加载项页面，刷新后即可看到可用的Add-on
